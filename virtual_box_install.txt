

Довольно шустро работает вроде бы, графика сервера в том числе -
   при условии что сам X-сервер не на вбоксе - а на хостовой тачке.


Я юзаю cygwin x-server, установка:

cygwin/usr/X11R6/bin  
  startxwin.bat
  xhost.exe+

На вбоксе: установить вместо NAT-а Host Interface создав виртуальный адаптер.

После запуска кноппикса:
   ifconfig eth0 up
   ifconfig eth0 inet 169.254.149.38
   # IP - IP вмбокса.
   sudo passwd knoppix
   # вбить новый пароль для юзера

После этого вмбокс доступен по ssh/sftp для закачки файлов
(можно как-то включить хостовый интернет, но нах трах).

теперь через ssh запускаем сервак и форвардим x-трафик на хост:
  2008/runsim.bat
  тут надо прописать IP хоста (из виртуального адаптера, смотреть через ipconfig /all)

запускаем контроллер
  dummy_controller/runsimu.bat
  тут IP или хостнейм вбокса тоже вбить надо.

Эврика. К вбоксовой консоли даже не прикасаемся, все с хоста ;)
